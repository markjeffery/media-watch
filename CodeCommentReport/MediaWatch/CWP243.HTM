    InitialiseGrid Initialise Grid. This is done the first time the page is loaded, or when the channel changes. It gets the range of dates for transmission (startTrans to endTrans) and sets parameters for the first view of the data. This is always the most recent records, so it asks for the previous set of data with date < endTrans  
    setupCmd Setup Query command. This sets up a query command for either "chunk" or "asrun" table. and either "next" or "prev" for a particular channel and the top nrecs

 
    UpdateDateRange    
    SelectMiddle Make a selection in the "middle" row of the datagrid. This is done when the calendar control is clicked or the next page button is clicked, and no "particular" row is selected.

 
    FillGrid Fill Grid according to current state. This can happen from InitialiseGrid if this is the first time the page has been shown. This can happen from previous / next page commands or as a consequence of previous / next rec commands.

 
    enableRecButtons Enable or disable previous / next rec buttons. This needs to be done after selection is made. Cell 6 of the selected item in the datagrid (an unformatted start_tc) is compared against startTrans for the prev button and endTrans for the next button. Also, next date is queried, and stored on web page, so that MediaPlayer knows how to get the next chunk.
