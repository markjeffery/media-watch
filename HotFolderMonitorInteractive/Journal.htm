  • Configured SDK Quickstarts, tutorials and samples
  • Created MediaWatch database tables asrun, channels and chunks.
  • Configured access to MSDE for ASPNET user for databases. (In Windows Server 2003 ASP.NET User Account is 'NT AUTHORITY\NETWORK SERVICE' See Config Details for more details) 
    • osql -E -S (local)\NetSDK -Q "sp_grantlogin 'NT AUTHORITY\NETWORK SERVICE'" osql -E -S (local)\NetSDK -d MediaWatch -Q "sp_grantdbaccess 'NT AUTHORITY\NETWORK SERVICE'"
    • osql -E -S (local)\NetSDK -d MediaWatch -Q "sp_addrolemember 'db_owner', 'NT AUTHORITY\NETWORK SERVICE'"
  • Created MediaWatchPopulate application to populate tables with definable amounts of data, so that testing can be done with "real" amounts of data.
